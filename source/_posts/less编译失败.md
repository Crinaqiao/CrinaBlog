**1.先卸载**

```
npm uninstall less-loader less
npm uninstall less-loader less
```

**2.安装**
```
npm install less-loader@4.1.0 less@3.9.0 -D
npm install less-loader@4.1.0 less@3.9.0
```

之后就可以使用了
因为Vue是2.X的版本
